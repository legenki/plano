Keyboard Shortcuts

Vertices Mode (1):
• Click empty space — Add vertex
• Drag vertex — Move vertex
• Double click vertex — Toggle connections
• Scroll over vertex — Change radius
• Backspace/Delete — Delete vertex

Glyphs Mode (2):
• Double click empty space — Create new glyph
• Drag — Move selected glyph
• Scroll over vertex — Scale glyph
• Shift + scroll — Rotate glyph around vertex
• Backspace/Delete — Delete selected glyph

Scene Mode (3):
• Drag — Pan camera
• Scroll — Zoom camera
• Shift + scroll over vertex — Rotate scene
• Shift + drag — Adjust grid pattern offset
• Shift + scroll on background — Rotate grid pattern
• Press X — Toggle pattern grid
• Backspace/Delete — Clear all glyphs
